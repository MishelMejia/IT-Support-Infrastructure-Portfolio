## Projects

### Enterprise Identity & Access Management System – Northbridge
**Technologies:** Active Directory, Windows Server 2022, PowerShell, RBAC, Group Policy  

This project simulates an enterprise identity and access management environment. It includes a structured Active Directory domain with departmental OUs, role-based access control, and automated user provisioning using PowerShell.

Key highlights:
- Active Directory domain deployment (`northbridge.local`)
- Role-Based Access Control using security groups
- Bulk user provisioning with PowerShell and CSV imports
- Group Policy security configuration
- Identity lifecycle management simulation

🔗 Project repository:  
https://github.com/yourusername/ActiveDirectory-Enterprise-IAM-Lab

### Active Directory Administration Lab
**Technologies:** Active Directory, Windows Server 2022, Group Policy, RSAT  

This lab focuses on Active Directory administration and security. It demonstrates how delegated administration and least-privilege access models are implemented in enterprise environments.

Key highlights:
- Domain controller and domain-joined workstation setup
- Organizational Unit and group management
- Delegated administration scenarios
- Group Policy security configuration
- Administrative access testing using RSAT

🔗 Project repository:
